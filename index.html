Практические задачи

  • У Вас есть большой список номеров банковских карт. Необходимо написать программу, которая на старте будет читать этот список и уметь по заданному номеру карты определять его наличие в списке. Поиск номера в списке реализовать наиболее эффективным способом.
  • Соответствие номера телефона сотовому оператору определяется по таблице def-кодов, которая выглядит следующим образом (файл CSV формата): 
    
	909,97,beeline
	985,312-315,mts
	...
    Первая колонка - три цифры - это def код. Вторая колонка задает диапазон и может иметь два формата:
    1. Префикс диапазона. Например, 97 - все номера, которые начинаются на 97
    2. Диапазон префикса. Например, 312-315 - все номера, которые начинаются с 312, 314 и 315
    Ваша программа должна уметь определять корректность исходных данных - не должно быть пересечений диапазонов - и по заданному номеру телефона наиболее эффективно определять оператора.
